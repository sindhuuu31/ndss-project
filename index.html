2. Critical Analysis Parameters

  • Packet Count: High sudden surges (>5000/s) signify volumetric flood attacks.
  • Packet Size: Legitimate web traffic varies. Constant fixed sizes (like exactly 64 bytes) strictly indicate automated attack tools.
  • Entropy: Low entropy indicates highly repetitive, non-random payload bytes. High entropy implies natural encrypted user data.

3. Intelligent Threat Explainer

Using a Random Forest machine learning model, the system calculates exact probabilities of malicious behavior. Instead of just filtering and blocking traffic silently, the AI generates a detailed explanation of exactly why the decision was made, analyzing WHAT occurred and HOW it reached the decision, fully empowering network administrators.
